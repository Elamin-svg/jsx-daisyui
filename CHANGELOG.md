# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-17

### Added

- Initial release of `jsx-daisyui`
- 60 components across 6 categories: Forms, Data Display, Navigation, Feedback, Layout, Effects
- CLI command `npx jsx-daisyui add <component>` to copy components into your project
- Full TypeScript support with type definitions
- Compatible with daisyUI v5 and Tailwind CSS v4
- React 18+ support
